## Domain Model

<Callout type="info" title="Objectif">
Le Domain Model décrit les entités métiers du club d’arts martiaux.
</Callout>

Les principales entités sont :

- User
- Course
- Category
- Post
- Media

<Callout type="tip">
La modélisation du domaine est toujours réalisée **avant le backend**.
</Callout>
